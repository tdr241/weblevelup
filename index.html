Название элемента, пример

    fdv

  • code - элемент для отображения кода из программы. Пример: hello,world
    my name is Sasha
  • li - элемент для создания таблицы, используется совместно с "ul" - маркерованный список и "ol" - нумированнаый все эти элементы - блочные
  • em - курсивное написание. Пример: Невоможно - слово из словаря глупцов.
  • addrees - этот элемент говорит браузеру о том, что указан адресс.Пример:
    Leninskiy prospect 161
  • br - перенос строки
  • blockquote - блочный элемент , большая цитата, выделяется как блок.
  • q - строчный элемет, цитата. Выделяется ковычками
  • hr- горизонтальная линия
  • strong - выделяет текст жирно. Пример: BLACK and White
  • pre - элемент для отформатированного текста, браузер отобразит его так, как я напечатал. (не нужно переносить с помощью br и тд.)
  • b - жирное выделение
