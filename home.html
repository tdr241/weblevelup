Название элемента, пример

  1. code - элемент для отображения кода из программы. Пример: hello,world
    my name is Sasha
  2. li - элемент для создания таблицы, используется совместно с "ul" - маркерованный список и "ol" - нумированнаый все эти элементы - блочные
  3. em - курсивное написание. Пример: Невоможно - слово из словаря глупцов.
  4. addrees - этот элемент говорит браузеру о том, что указан адресс.Пример:
    Leninskiy prospect 161
  5. br - перенос строки
  6. blockquote - блочный элемент , большая цитата, выделяется как блок.
  7. q - строчный элемет, цитата. Выделяется ковычками
  8. hr- горизонтальная линия
  9. strong - выделяет текст жирно. Пример: BLACK and White
  10. pre - элемент для отформатированного текста, браузер отобразит его так, как я напечатал. (не нужно переносить с помощью br и тд.)
  11. b - жирное выделение
